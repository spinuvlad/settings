Start Tutorial: https://www.youtube.com/watch?v=Yeh4ilpiWTY&t=0s

1. Control Panel > Programs and Features > Turn Windows features on or of > Windows Subsystem for Linux > Ok > Restart

2. Install Ubuntu from Microsoft Store

3. Download and Run *.reg file from https://gist.github.com/agragregra/0945725f5b09727bdda77aca51311992

4. Run "Simple" or "Advanced" command in Ubuntu bash from: https://gist.github.com/agragregra/9e5741c53714ed86c7fe96f9a33921ff

5. Config Git:
		git config --global user.name "Name Family"
		git config --global user.email "my_email@domain.com"
		git config --list (verifica configurarile)

6. install sublime text3

	6.1 Preference --> Settings (copy/paste):
		"margin": 2,
		"fold_buttons": false,
		"tab_size": 2,
		"highlight_line": true,
		"line_padding_top": 1,
		"auto_complete": false,

	6.2 Preference --> Key Bindings [aranjeaza codul frumos] (copy/paste):
		{ "keys": ["alt+shift+f"], "command": "reindent" }, 

	6.3 CTRL + SHIFT + P --> install package
		1.6.1. emmet
		1.6.2. sass (View --> Syntax --> Open all with... --> sass)
		1.6.3. jade
		1.6.4. gist
		1.6.5. BracketHighlighter
		1.6.6. AutoFileName
		1.6.7. Color Highlighter
		1.6.8. Goto-CSS-Declaration   
			---gaseste clasa data in html in css sau sass, click dreapta pe class in html si alege "Go to css declaration" 											
			--> combinatia de taste poate fi observata in Preferance -->Package Settings  -->Goto  -->Key - Default:  "super=tasta windows"

	6.4 View --> Layout --> Columns 2
