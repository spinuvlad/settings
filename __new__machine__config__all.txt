Start Tutorial: https://www.youtube.com/watch?v=Yeh4ilpiWTY&t=0s

1. Control Panel > Programs and Features > Turn Windows features on or of > Windows Subsystem for Linux > Ok > Restart

2. Install Ubuntu from Microsoft Store

3. Download and Run *.reg file from https://gist.github.com/agragregra/0945725f5b09727bdda77aca51311992

4. Run "Simple" or "Advanced" command in Ubuntu bash from: https://gist.github.com/agragregra/9e5741c53714ed86c7fe96f9a33921ff

5. Config Git:
		git config --global user.name "Name Family"
		git config --global user.email "my_email@domain.com"
		git config --list (verifica configurarile)

6. install sublime text3
		
	6.1 Preference --> Settings (copy/paste):
		"margin": 2,
		"fold_buttons": false,
		"tab_size": 2,
		"highlight_line": true,
		"line_padding_top": 1,
		"auto_complete": false,

	6.2 install package:
		emmet
		sass (View --> Syntax --> Open all with... --> sass)
		jade
		gist
		BracketHighlighter
		Color Highlighter

	6.3 View --> Layout --> Columns 2

7. install visual studio code

8. clone and install ahk: https://github.com/spinuvlad/ahk.git
